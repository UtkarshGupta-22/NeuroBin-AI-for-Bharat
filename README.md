NeuroBin is an AI-powered smart waste segregation system that integrates edge AI, embedded automation, and IoT monitoring to improve waste management efficiency.

## Key Features
- AI-based biodegradable vs non-biodegradable classification
- Real-time inference (<500ms)
- Automated servo-based segregation
- Fill-level monitoring using ultrasonic sensors
- Cloud-ready IoT architecture
- Future generative AI analytics integration

## Architecture
Edge AI on Raspberry Pi 4 using TensorFlow Lite.
ESP32-CAM for image capture.
HC-SR04 sensors for fill-level detection.
Servo motors for mechanical actuation.
